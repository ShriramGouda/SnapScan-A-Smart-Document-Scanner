Getting Started with SnapScan AI

Welcome to SnapScan AI! Follow these steps to get started:

  1. Create an account - Sign up to save your documents and access advanced features
  2. Upload your first document - Use the Document Scanner section to upload or capture a document
  3. Process and extract text - Let our AI extract text from your document with high accuracy
  4. Use advanced features - Try summarization, translation, or paper evaluation
  5. Export your results - Download or save your processed documents

For more detailed instructions on specific features, select the relevant category above.

Using the Document Scanner

The Document Scanner allows you to digitize physical documents using your device's camera or by uploading image files.

Uploading Documents

  • Click the upload area or drag and drop files
  • Supported formats: JPG, PNG, PDF
  • For best results, use clear, well-lit images

Using the Camera

  • Click "Start Camera" to activate your device camera
  • Position the document within the frame
  • Click "Capture" to take a photo
  • Use "Auto-Enhance" to improve image quality

Processing Documents

  • Click "Process Document" to extract text
  • View results in the Extracted Text tab
  • Use AI Summary for condensed versions
  • Translate to other languages if needed

Paper Evaluation System

Our AI-powered paper evaluation system helps educators automatically grade answer sheets with detailed analysis.

How to Use

  1. Upload the question paper
  2. Upload the student's answer paper
  3. Upload the evaluation scheme (optional but recommended)
  4. Click "Evaluate Papers" to start analysis
  5. Review results across multiple tabs

Evaluation Features

  • Keyword Matching - Checks for key concepts in answers
  • Plagiarism Detection - Identifies copied content
  • Grammar Check - Finds language errors
  • Detailed Evaluation - Comprehensive answer analysis
  • Comparison Analysis - Compares with expected answers

Troubleshooting Common Issues

Camera Not Working

  • Ensure camera permissions are granted
  • Try refreshing the page
  • Check if another app is using the camera
  • Try the "Switch Camera" button for alternative cameras

Poor Text Recognition

  • Use better lighting when capturing documents
  • Ensure the document is flat and fully visible
  • Try the "Auto-Enhance" feature
  • For handwritten text, ensure clear handwriting

Slow Processing

  • Large documents may take longer to process
  • Check your internet connection
  • Try reducing image quality if speed is critical

Export Issues

  • Ensure you have storage permissions
  • Check available storage space on your device
  • Try a different export format
